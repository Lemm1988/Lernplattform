# News-System - Dokumentation

**Letzte Aktualisierung:** 27. Januar 2025

---

## 📋 Übersicht

Das News-System ermöglicht die Verwaltung und Anzeige von News-Artikeln.

---

## 🎯 Hauptfunktionen

- Artikel erstellen/bearbeiten/löschen
- Kategorien verwalten
- Tags verwalten
- Medien-Upload
- Rich-Text-Editor (TinyMCE)
- Veröffentlichungsstatus (draft/published)
- Featured-Artikel
