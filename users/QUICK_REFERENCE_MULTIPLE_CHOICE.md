# Quick Reference - Multiple-Choice Questions

## At a Glance

### Question Types

| Type | Symbol | Selection | Scoring |
|------|--------|-----------|---------|
| **Single-Choice** | ○ Radio buttons | One answer only | All or nothing |
| **Multiple-Choice** | ☐ Checkboxes | Multiple answers | Proportional |

### How to Answer

#### Single-Choice Questions
1. Read the question carefully
2. Click ONE radio button (○)
3. Click "Next" or "Submit"

#### Multiple-Choice Questions
1. Read the question carefully
2. Click ALL correct checkboxes (☐)
3. Avoid incorrect options
4. Click "Next" or "Submit"

### Scoring System

#### Single-Choice
- ✅ **Correct answer** = Full points
- ❌ **Wrong answer** = 0 points

#### Multiple-Choice
- ✅ **All correct selected** = Full points
- ⚠️ **Partially correct** = Partial points
- ❌ **Wrong selections** = Point deduction
- ⭕ **No selection** = 0 points

### Tips for Success

#### General Tips
- Read questions completely
- Look for keywords
- Take your time
- Review your answers

#### Multiple-Choice Tips
- Select ALL correct answers
- Avoid wrong answers
- When unsure: select fewer rather than more
- Use process of elimination

### Understanding Results

#### Result Symbols
- ✅ = Correctly selected
- ❌ = Incorrectly selected  
- ➖ = Correct answer not selected

#### Example Result
```
Question: Which are programming languages?
Your answers: ✅ PHP, ❌ HTML, ✅ JavaScript, ➖ Python
Correct answers: ✅ PHP, ❌ HTML, ✅ JavaScript, ✅ Python
Score: 1.5 out of 2 points
```

### Code Examples

Some questions include code examples:
```php
<?php
$x = 5;
$y = 10;

?>
```

Code is safely formatted and highlighted for better readability.
